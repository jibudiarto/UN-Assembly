# UN-Assembly

Using clustering analysis on 69 years of UN General Assembly voting data (1946-2015), this study groups countries by their voting patterns across 6 resolution categories: Israeli-Palestinian Conflict, Colonialism, Economic Development, Human Rights, Nuclear Weapons, and Arms Control and Disarmament.
